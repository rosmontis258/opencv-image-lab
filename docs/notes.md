## 3. 当前存在的问题

# 3.1 效果问题
1. 二值化效果不理想，后续需要尝试 Otsu 或自适应二值化
2. 边缘检测结果不够清晰，后续需要调整 Canny 参数，并考虑使用模糊后的灰度图作为输入
3. 腐蚀和膨胀的效果过于明显，后续需要调整核大小和迭代次数

# 3.2 工程结构问题
1. 目前所有逻辑都堆在 main 中，未进行函数封装
2. 暂时无法按需调用不同处理流程
3. 尚未实现图像尺寸调整
4. 尚未实现处理结果保存和统一展示

# 3.3 认知问题
1. 对不同图像处理算法适合处理哪类输入图还不够清楚
2. 对二值图、灰度图、彩色图在不同算法中的使用场景还需要继续理解

完成一个基础图像处理实验台，能够对同一张图片完成：
- 预处理
- 二值化对比
- 边缘检测
- 基础形态学处理
并具备初步的模块化调用能力。

## 6. 下一步执行顺序
1. 先整理代码结构，拆分 main
2. 再补图像尺寸调整和进阶二值化
3. 然后补开闭运算和结果对比
4. 最后考虑滑块调参和界面优化
